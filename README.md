## Central thesis

The transition from **written culture** to **digital culture** is at least as significant
as the transition from oral to written culture. It changes the mathematical structure
of social space (the Topological Revolution), the ergodic properties of information,
and the fitness of every institution built for the written era.

## File naming convention

```
sol-000-*.md    Reference files (lexicon, maps, logic)
sol-000b-*.md   Pre-foundational articles (2020-2024, philosophical roots)

```

All files are self-contained with internal XML metadata.
No directory hierarchy — flat structure optimized for LLM context injection.
