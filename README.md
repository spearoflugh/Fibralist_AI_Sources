# Fibralist_AI_Sources
Contains the sources of posts from https://thefibralist.substack.com/ annotated to optimize AI context building. https://thefibralist.substack.com/

The goal is to provide the raw texts with meta-data, identifiers and semantical descriptions of the article published. It is thought as being the successor of "book" or blog way of building a piece of information with the explicit idea that information is going to be accessed via LLMs.
